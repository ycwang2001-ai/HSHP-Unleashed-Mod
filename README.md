High Seas, High Profits! Unleashed Mod

Added three new game options:

1. Build Shops & Dwellings
- All cities start with zero population with no dwellings and businesses pre-built.
- No AI traders.
- Player start with extra money, and 250 stockpile of all goods plus extra building materials in home city.
- Add 100 repulation and 50 mayor respect per dwelling placed.

2. FB's Changes
- Businesses can only be built in towns with a bonus.
- Cities spawns with only one bonus good (raw only, no processed goods) but can get additional thru mayor missions.
- 5x the cost of bonus to goods mayor mission but made duration permanent.
- Population capped at dwelling capacity, details in Trader's Hall.
- Occupancy-based happiness instead of homelessness.
- Lowered reputation thresholds with 2x gain, also gain mayor respect from selling goods.
- No mayor elections.
- AI does not auto complete mayor missions.

3. Black Death
- All cities start with zero population with no dwellings and businesses pre-built.
- No AI traders.
- Player start with extra money, and 250 stockpile of all goods plus extra building materials in home city.
- Add 100 repulation and 50 mayor respect per dwelling placed.
